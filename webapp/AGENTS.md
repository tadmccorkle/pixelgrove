# Frontend Agent Guide

## Overview

Single-page application with a clean, minimal UI built with:

- **Runtime**: Bun
- **Framework**: React with TypeScript
- **Styling**: Tailwind CSS
- **Components**: Shadcn/Radix UI

## Commands

All commands should be executed from this directory (`webapp/`).

- **Install**: `bun install`
- **Build**: `bun install && bun run build`
- **Run**: `bun install && bun dev`
- **Lint**: `bun lint`
- **Format**: `bun format`

## Development Notes

- The backend API runs at `http://localhost:4815` in development.
